This repository implements a simulation and modeling framework for **Reconfigurable Intelligent Surface (RIS)**-assisted IoT networks operating under **time-varying wireless channels**.  
It aims to study and mitigate signal degradation in limited feedback conditions by leveraging optimized configurations and learning-based models.

This work is an extension of our team ongoing research on **RIS-assisted 6G IoT networks** and corresponds to the findings discussed in the paper:

> ğŸ“„ [Attention-Guided Encoder-Decoder Design for Efficient RIS Phase Configuration in Time-Varying IoT Networks](https://ieeexplore.ieee.org/abstract/document/11168457)  
> _IEEE Transactions on Artificial Intelligence, 2025._
